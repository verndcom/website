# Vernd LLC Static Site

## Deploy to GitHub Pages (custom domain: vernd.com)
1. Create a repo and push these files to the repo root.
2. Repo Settings -> Pages:
   - Source: Deploy from a branch
   - Branch: main / (root)
3. Set Custom domain to vernd.com and enable HTTPS after DNS is verified.

## Files
- index.html, about.html, services.html, careers.html, privacy.html, terms.html
- assets/style.css, assets/site.js
- assets/logo-light.png
